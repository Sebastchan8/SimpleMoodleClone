# Moodle Clone

A Moodle-like platform for managing courses and participants. Features include user authentication, course creation, and participant enrollment.

## Technologies Used
- HTML
- CSS
- PHP
- JavaScript
- MySQL

## Features
- User authentication system.
- Create, manage, and enroll in courses.
- Responsive design for web accessibility.

2. Clone the repository:
   ```bash

   ```
